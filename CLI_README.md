# Monad BFT CLI Helper

Interactive console helper for the **Monad BFT** README.

Styled for Windows CMD / terminal, it uses `colorama` for colored output and
provides a convenient menu with:

- Overview
- Getting Started
- Using Docker
- Docker Build Cache Tip
- Using Pre-built Images
- Test RPC Connection
- Run via Cargo: Consensus Client
- Run via Cargo: JsonRpc Server
- Install Dependencies
- Settings
- About

## Install CLI dependencies

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```

After starting, you will see the Monad BFT ASCII logo and the main menu.
Control is via entering the menu number and pressing Enter.
